✨ 기능 가이드 / Feature Guide

🎭 Avatar System 완료

URL 파라미터로 아바타 유형 지정:

?avatar=shiba (강아지, 기본값)
?avatar=robot (로봇)
?avatar=none (카메라만)

✋ Drag & Transform 완료 admin only

로그인한 관리자만 사용 가능. 3D 오브젝트를 드래그하거나:

  • 드래그: 마우스 드래그로 위치 이동
  • Transform 패널: ✏️ 드래그패널에서 scale/rotate 슬라이더로 세밀한 조정
  • Viewers는 드래그 불가 (읽기 전용)

📍 Location Presets 완료

방 안의 특정 위치를 저장하고 빠르게 이동:

  • 관리자가 방에서 "💾 Save Pos" 클릭 → 이름 입력 → 프리셋 저장
  • 모든 사용자가 드롭다운으로瞬間 텔레포트
  • is_default=true인 프리셋이 입구( spawn point)가 됨
  • URL: ?preset=<preset_id>

🗺️ Mini-map 완료

우측 상단 미니맵:

  • 🟢 점 = 내 위치 + 방향
  • 🔵 점 = 프리셋 위치
  • ⭐ (골드) = 기본 프리셋
  • 클릭하면 기본 프리셋으로 텔레포트

🚧 Boundary Lock 완료

Room模型的 boundary_min/max_* 필드로 플레이 영역 제한:

  • 바닥(Y<0) 이하로 내려가지 않음
  • 벽 범위(X/Z) 밖으로 나가지 않음
  • Room 편집에서 boundary 값 조정 가능

✂️ Gaussian Splat Cutout 완료

splat 요소 페이지에서:

  • cutout_scale: 잘라낼 영역 크기 (예: 0.5 0.5 0.5)
  • cutout_position: 잘라낼 위치 오프셋 (예: 0 0 0)
  • 체크박스로 컷아웃 켜기/끄기

🌐 Webpage Embed 완료

웹페이지를 3D 벽면에 임베드:

  • Webpage 모델에서 webpage_url 입력
  • width, height로 크기 조절
  • YouTube/Twitch 임베드 URL도 가능
  • 일부 사이트는 X-Frame-Options으로 차단될 수 있음

🎬 Video Background 완료

벽면에 동영상 배경 설정:

  • Wall模型的 video_url 필드에 URL 입력
  • mp4, webm 또는 HLS(.m3u8) 스트림 가능
  • 동영상 파일이면 직접 재생, RTSP는 스트리밍 서버 필요

🤚 Hand Tracking 완료

Quest 3 / Vision Pro에서:

  • "🤚 Hand Tracking" 버튼이 자동으로 표시
  • 클릭 → 손 추적 활성화
  • 다른 사용자들도 초록색 손 오브젝트로 보임

🥽 AR Passthrough 완료 admin only

관리자만 사용 가능:

  • Quest 3 / ARCore 기기에서 "🥽 AR Passthrough" 버튼 클릭
  • 현실 세계 카메라 피드 위에 가상 오브젝트 표시
  • 위치/스케일/회전 조정 후 실제 환경에서 확인 가능

🤖 LLM Auto-Layout 완료 admin only

자연어로 이미지 배치를 지시:

  • 설정: 좌측 메뉴에서 LLM Config → provider, api_base, api_key, model 입력 → is_active 체크
  • 지원 provider: minimax, openai, openrouter, anthropic
  • 사용: Room 안에서 관리자 패널 → "Auto Layout" 버튼 → 프롬프트 입력
  • 예시: "Place traditional paintings on the left wall, modern art on the right"
  • API: POST /api/auto-layout → POST /api/apply-layout

🎭 Room Effects (LLM) 완료 admin only

자연어로 시각/오디오 이펙트 트리거:

  • 설정: LLM Config가 활성화되어 있어야 함 (위 참조)
  • 사용: Room 안에서 "Auto Effect" 버튼 → 프롬프트 입력
  • 예시: "Add glitter particles around the main artwork"
  • 이펙트 유형: glitter, spotlight, ambient, fog, sound, pulse, color_shift, shake, fade
  • 이펙트를 직접 관리하려면 좌측 메뉴 Room Effect에서 수동 생성/삭제 가능
  • API: POST /api/auto-effect, POST /api/clear-effects

🎤 Whisper Voice Transcription 완료

음성 채팅을 텍스트로 실시간 전사:

  • 설정: 좌측 메뉴에서 Whisper Config → provider, api_base, api_key 입력 → enabled 체크
  • 지원 provider: openai (whisper-1), minimax, local
  • language를 비워두면 자동 감지 (또는 ko, en, ja 등 지정)
  • 결과: 음성 채팅 시 아바타 위에 말풍선으로 텍스트 표시 (5초간)
  • API: POST /api/transcribe

📍 AR Markers 완료

물리적 마커 → 가상 위치 매핑 (AR.js 기반):

  • 설정: 좌측 메뉴에서 Marker → room 선택, marker_type/value 입력, target_position 설정
  • marker_type: hiro (내장), pattern (커스텀 인쇄), image (NFT 이미지)
  • target_preset을 설정하면 LocationPreset으로 직접 링크 가능
  • 사용: /room?room_id=...&ar=marker → 카메라로 마커 스캔 → 가상 위치로 이동
  • Vision Pro는 카메라 접근 불가 → iPhone companion 모드 사용: /ar-companion?room_id=...

🔄 Auto-refresh 완료

Wall 페이지에서:

?refresh=30 → 30초마다 컨텐츠 변경 확인 후 페이지만Reload
변경 없으면Reload 안 함 (ETag 비교)

🔐 Authentication 완료

드래그/변환은 로그인한 관리자만 가능:

  • URL에 ?admin=1 파라미터는 더 이상 사용 안 함
  • 실제 admin 계정으로 로그인 필요
  • /kwanri → 로그인

⚙️ LLM & Whisper 빠른 설정 / Quick Setup

LLM Config (자동 레이아웃 & 이펙트)

  1. 좌측 메뉴에서 LLM Config 클릭
  2. Create 클릭 (또는 기존 설정 편집)
  3. 필수 입력:
    • provider: minimax, openai, openrouter, 또는 anthropic
    • api_base: API 엔드포인트 (예: https://api.openai.com/v1)
    • api_key: API 키
    • model: 모델명 (예: gpt-4o, MiniMax-M2.7)
  4. is_active 체크 → 저장
  5. Room에서 "Auto Layout" 또는 "Auto Effect" 버튼 사용 가능

Whisper Config (음성 전사)

  1. 좌측 메뉴에서 Whisper Config 클릭
  2. Create 클릭 (또는 기존 설정 편집)
  3. 필수 입력:
    • provider: openai, minimax, 또는 local
    • api_base: API 엔드포인트 (예: https://api.openai.com/v1)
    • api_key: API 키
  4. enabled 체크 → 저장
  5. Room에서 음성 채팅 시 자동 전사 + 아바타 위 말풍선 표시
💡 참고: API 키는 보안을 위해 목록 화면에서는 숨겨집니다. 편집 화면에서만 확인/수정 가능합니다.

📹 라이브 스트리밍 / Live Streaming

IP 카메라 (RTSP) 설정 필요

  1. Wall 페이지에서 관리자로 로그인
  2. 하단 우측 📹 Stream 버튼 클릭
  3. RTSP URL 입력 (예: rtsp://camera:554/stream)
  4. ▶ Start RTSP 클릭
  5. Status에 표시된 URL 복사 (예: /stream/playlist/wall1)
  6. Wall模型的 video_url에 붙여넣기

폰 카메라 라이브 설정 필요

  1. Wall 페이지에서 관리자로 로그인
  2. 하단 우측 📹 Stream 버튼 클릭
  3. 📹 Start Phone Camera 클릭
  4. 카메라 권한 허용
  5. 화면의 QR코드나 URL로 접속하는 다른 사용자도 같은 영상을 볼 수 있음
  6. ⏹ Stop Camera로 중지

YouTube / Twitch

Webpage 모델 사용:

webpage_url = https://www.youtube.com/embed/LIVE_STREAM_ID?autoplay=1&mute=1

💡 팁 / Tips

  • 미니맵 없애기: 미니맵은 자동으로 표시되며 클릭으로 기본 위치로 텔레포트
  • 위치 조정: 드래그로 대략적 위치 → Transform 패널에서 좌표 직접 입력
  • 여러 브라우저 테스트: 시크릿 모드에서 일반 사용자 뷰 확인
  • 갤러리 미리보기: avatar=none으로 3D 씬만 보고 아바타 숨기기
  • RTSP 안 될 때: IP 카메라가同一 네트워크에 있는지 확인, URL 형식 확인
  • 카메라 스트리밍 안 될 때: HTTPS 필요 (cloudflaredunnel은 HTTPS 제공)
